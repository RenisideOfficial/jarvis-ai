## ✨ Features

### 🎙️ **Voice-Activated UI Transformations**

- **"QUANTUM"** — Reality distortion field with purple energy effects
- **"STEALTH"** — Tactical camouflage mode with reduced emissions
- **"OVERDRIVE"** — Hyper-performance mode with glitch effects

### 🖥️ **Real-Time System Monitor**

- CPU, Memory, GPU, and Network utilization with animated bars
- Uptime counter and process tracking
- Dynamic power draw visualization

### 📡 **Holographic Radar System**

- Rotating radar sweep with glowing trail
- Fading contact dots that respawn naturally
- Range rings and frequency readouts

### 💬 **AI-Powered Chat Interface**

- Powered by Claude Haiku AI
- Secure proxy server architecture (API key stays safe on backend)
- Real-time response metrics (tokens, latency, confidence)

### 🔊 **Immersive Audio Experience**

- Continuous ambient hum (60Hz "void sound")
- Distinct sound effects for:
  - Button clicks
  - Message send/receive
  - Voice activation
  - System boot sequence

### 🌟 **Visual Effects**

- Animated starfield background
- Scanline overlay for CRT feel
- Custom cyberpunk cursor with hover animations
- Corner accent decorations

### 🗺️ **Additional Features**

- Geolocation tracking
- System modules status panel
- Alert feed with real-time notifications
- Performance sparkline graphs

## 🚀 Quick Start

### Prerequisites

- **Node.js** (v16 or higher)
- **Claude API Key** (from Anthropic)
- Modern web browser with Web Speech API support

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/RenisideOfficial/jarvis-ai.git
   cd jarvis-ai
   ```
